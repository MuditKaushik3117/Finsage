# 💹 FinSage

### Invest Smarter. Grow Faster.

FinSage is an AI-powered investment advisory platform that helps users make informed financial decisions by generating personalized portfolio recommendations based on their investment goals, risk appetite, and financial profile.

Built as a modern fintech web application using React and Vite, FinSage provides an interactive dashboard, portfolio recommendations, market insights, and portfolio rebalancing suggestions.

---


# ✨ Features

- 🤖 AI-inspired Investment Recommendations
- 📊 Interactive Portfolio Dashboard
- 📈 Portfolio Allocation Visualization
- 🌍 Market Outlook & Insights
- ⚖️ Portfolio Rebalancing Suggestions
- 📝 Investor Risk Assessment
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Modern UI using Tailwind CSS

---

# 🧠 How It Works

1. User completes a financial risk assessment.
2. FinSage analyzes:
   - Risk Tolerance
   - Investment Horizon
   - Investment Goals
3. AI-inspired logic recommends a diversified portfolio.
4. Users can explore:
   - Portfolio Dashboard
   - Market Trends
   - Rebalancing Suggestions

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/MuditKaushik3117/Finsage.git
```

Go into the project

```bash
cd Finsage
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
